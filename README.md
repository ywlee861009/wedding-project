# 💍 Wedding Project: Our Special Day (TBD)

> **"우리만의 특별한 결혼식을 위한 웨딩 웹사이트 & 모바일 청첩장 프로젝트"**

이 프로젝트는 신랑과 신부의 소중한 결혼식을 위해 제작되는 맞춤형 웨딩 프로젝트입니다. 모바일 청첩장 기능부터 RSVP 관리, 지도 안내 등 결혼식에 필요한 다양한 기능을 담을 예정입니다.

## 🎮 게임 레퍼런스 (Game Reference)

본 프로젝트는 **[Coastal World](https://coastalworld.com/)**의 조작감, 색감, 인터랙션 방식을 핵심 벤치마크로 삼고 있습니다.
- **조작**: 직관적인 이동 및 카메라 워킹.
- **비주얼**: 밝고 화사한 파스텔 톤의 Low-poly 3D 스타일.
- **경험**: 누구나 쉽게 즐길 수 있는 접근성 높은 인터랙티브 웹 경험.

## 🛠️ 예상 기술 스택 (Potential Tech Stack)

- **UI Framework**: React (Next.js) 또는 Compose Multiplatform
- **Styling**: Vanilla CSS / Tailwind CSS
- **Data**: Firebase (Realtime Database / Firestore) - RSVP 및 방명록 저장용

---

## 🌟 주요 기능 (Core Features)

1.  **메인 초대장**: 신랑, 신부의 사진과 함께 특별한 초대 문구 제공.
2.  **웨딩 갤러리**: 소중한 순간들을 담은 고화질 웨딩 사진첩.
3.  **위치 및 지도**: 카카오맵/네이버지도 연동을 통한 식장 위치 안내 및 교통 정보.
4.  **RSVP**: 참석 여부 및 인원 확인을 위한 실시간 데이터 수집.
5.  **방명록**: 축하 메시지를 남길 수 있는 실시간 게시판.
6.  **계좌 번호 및 송금**: 축의금 전달을 위한 계좌 정보 및 카카오페이/토스 링크.
