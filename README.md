# 케로와 친구들 - 생존 게임

뱀파이어 서바이벌 / 탕탕특공대 스타일의 2D 탑다운 생존 게임.

## 캐릭터

- **영우**: 밸런스형 전사. 근접 + 원거리 공격.
- **문희**: 범위 공격형. 광역 마법으로 다수의 적을 처리.
- **케로**: 동반 펫. 영우/문희 옆에서 추가 투사체 발사.

## 게임 방법

- **이동**: WASD 또는 화살표 키 (모바일: 좌측 조이스틱)
- **공격**: 자동 — 가장 가까운 적을 향해 자동으로 공격
- **레벨업**: 적 처치 → 경험치 오브 획득 → XP 채우면 레벨업 → 능력 카드 3개 중 1개 선택
- **목표**: 최대한 오래 생존

## 실행 방법

```bash
python3 -m http.server 8080
# → 브라우저에서 localhost:8080 접속
```

## 기술 스택

- Vanilla JS (ES Modules)
- Canvas 2D API
- 외부 의존성 없음
