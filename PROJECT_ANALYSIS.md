# 💍 Wedding Project Analysis & Handover Report

이 문서는 다른 환경에서 프로젝트를 이어서 진행할 수 있도록 현재 프로젝트의 아키텍처, 구현 현황 및 기술 스택을 상세히 기록한 리포트입니다.

## 1. 프로젝트 개요 (Project Overview)
- **목표**: 3D 횡스크롤 게임 요소가 포함된 인터랙티브 모바일 청첩장 개발.
- **핵심 컨셉**: 2018년 첫 만남(독산역)부터 현재까지의 여정을 3D 공간에서의 이동과 시간 변화로 시각화.
- **주요 기술**: Android (Native Wrapper) + Three.js (3D Web Engine).

## 2. 시스템 아키텍처 (Architecture)

### 2.1 Android Wrapper (`/android`)
- **언어**: Kotlin
- **구현 특징**:
    - `WebView`를 통한 로컬 3D 콘텐츠(`web/index.html`) 로드.
    - **Chrome Remote Debugging**: PC 크롬 브라우저에서 `chrome://inspect`를 통해 디버깅 가능.

### 2.2 3D Game Engine (`/web`)
- **엔진**: Three.js (CDN 로드)
- **구조**:
    - `index.html`: Three.js 컨테이너 및 UI 오버레이.
    - `main.js`: 3D 씬(Scene) 관리, 렌더 루프 및 카메라 추적 로직.
    - `style.css`: 3D 캔버스 및 UI 레이아웃 스타일링.
    - `assets/`: 3D 모델(`.glb`), 텍스처 및 기타 리소스 저장소.

## 3. 주요 구현 기능 (Key Features Implemented)

### 3.1 3D 캐릭터 및 공간 시스템
- **Groom (신랑)**: 3D 모델 기반 주인공. WASD/방향키로 3D 공간 이동 및 점프 가능.
- **Side-scroller Camera**: 벨트스크롤 방식의 카메라 뷰. 캐릭터 이동 시 부드러운 원근감(Lerp)과 미세한 Z축 추적 기능 포함.
- **Environment**: 3D 나무, 가로등 등 입체적인 환경 오브젝트 배치 및 그림자(Shadow) 처리.

### 3.2 시간 및 환경 시스템
- **Dynamic Lighting**: 실시간 조명(Directional Light)을 통해 시간의 변화 연출 가능.
- **Atmospheric Fog**: 안개 효과를 통해 3D 공간의 깊이감과 우아한 분위기 조성.
